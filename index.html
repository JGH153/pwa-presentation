Progressive Web Apps
(PWA)

Buzzword

A collection of technologies/features that enable native like applications, with a seamless experience despite changing conditions
Let's break it down
Not a single thing
A series of new browser capabilities
What are we trying to solve with PWA?
The dream of the native app
Getting your app installed is impossible
PWA to the rescure
Enhancing and improving the user experience
PWA can remove the need for a seperate native application
Need smaller and faster websites
New definition
Make web applications appear more like traditional applications, or native mobile applications
What's included under the banner of PWA?
  • Responsive design
  • Progressive enhancements
  • Connectivity independent
  • App-like
  • Native features
  • Fresh/updated
  • Safe
  • Discoverable
  • Re-engageable (notifications)
  • Installable
  • Linkable
  • Geolocation
  • Fetch API (replaces XMLHttpRequest)
  • IndexedDB (for the SQLite guys)
  • Web Bluetooth
  • WebRTC (my last prez)
  • Battery API
  • Accelerometer API
  • Payment API
  • Vibration
  • VR/AR
  • Server side rendering
  • HTTP2
  • dns-prefetch
  • App Shell
  • Fluid animations
  • Near-instant loading
That's a LONG list
